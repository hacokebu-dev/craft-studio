마크다운(Markdown) 문법을 한눈에 확인하고, 바로 복사해서 테스트해 보실 수 있도록 정리해 드립니다. 아래 내용을 복사해서 마크다운 에디터(Notion, Obsidian, GitHub 등)에 붙여넣어 보세요.

---

## 1. 마크다운 렌더링 예시 및 코드

### 제목 (Headings)

# H1 제목

## H2 제목

### H3 제목

#### H4 제목

---

### 텍스트 강조 (Emphasis)

**굵게 쓰기 (Bold)**
_기울여 쓰기 (Italic)_
~~취소선 (Strikethrough)~~
<u>밑줄 (에디터에 따라 지원 여부 다름)</u>

---

### 목록 (Lists)

**순서가 없는 목록**

- 사과
- 바나나
- 소분류 (들여쓰기)

- 포도

**순서가 있는 목록**

1. 첫 번째
2. 두 번째
3. 세 번째

**체크리스트 (Task Lists)**

- [x] 완료된 할 일
- [ ] 미완료 할 일

---

### 인용문 및 구분선 (Quotes & Rules)

> 이것은 인용문입니다.
>
> > 중첩된 인용문도 가능합니다.

---

### 링크 및 이미지 (Links & Images)

[Google 바로가기](https://www.google.com)

---

### 코드 (Code)

`inline code`는 문장 중간에 넣을 수 있습니다.

**코드 블록 (Code Blocks)**

```python
def hello_world():
    print("Hello, Markdown!")

```

---

### 표 (Tables)

| 제목      | 내용      | 비고        |
| --------- | --------- | ----------- |
| 왼쪽 정렬 | 중앙 정렬 | 오른쪽 정렬 |
| 열 1      | 열 2      | 열 3        |

---

## 2. 복사 전용 전체 코드 블록

아래 박스 오른쪽 상단의 복사 버튼을 눌러 전체 내용을 한 번에 가져가세요.

````markdown
# 마크다운 테스트 페이지

## 1. 텍스트 스타일

이 문장은 **굵게** 표현됩니다.
이 문장은 _기울임_ 꼴입니다.
이 문장은 ~~취소선~~이 적용됩니다.

---

## 2. 목록 테스트

### 체크리스트

- [x] 마크다운 배우기
- [ ] 웹사이트에 적용하기

### 순서 없는 목록

- 항목 A
  - 하위 항목 A-1
- 항목 B

## 3. 인용 및 코드

> "지식은 공유될 때 가치가 있다."

문장 속에 `const code = 'test'`처럼 코드를 넣을 수 있습니다.

```javascript
function greet(name) {
  console.log("Hello, " + name + "!");
}
greet("User");
```
````
