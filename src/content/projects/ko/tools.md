<h1>Overview</h1>
<p>〈Tools〉는 출판 편집자의 업무 편의를 위해 제작된 툴킷 서비스입니다. 본 서비스는 출판 편집자 커뮤니티인 <a href="https://editorsroom.kr" target="_blank">에디터스 룸</a>의 요청으로 제작되었으며, 해당 커뮤니티의 도움을 받아 현직자들의 실제 목소리와 필요를 충실히 반영했습니다.</p>
</div>

<div class="text-narrow">
<h1>01. 문장 부호 복사기</h1>
<ul>
  <li>핵심 기능: 편집 실무에서 가장 빈도가 높은 9가지 문장 부호를 기본 제공합니다.</li>
  <li>커스텀 설정: 나만의 부호를 추가하고 관리(삭제)할 수 있습니다.</li>
  <li>편의성: 부호의 이름과 용도를 직관적으로 디자인하여 가독성을 높였습니다.</li>
  <li>개인정보 보호: 별도의 회원가입 없이 로컬 스토리지를 활용해 사용자 데이터를 안전하게 저장합니다.</li>
</ul>
</div>
<br>
<img src="/assets/tools_05.jpg" alt="5" />
<br>
<div class="text-narrow">
<h1>02. 책등 계산기</h1>
<ul>
  <li>정밀 계산: 종이 종류와 평량(g/㎡), 쪽수를 선택하여 책의 두께를 즉시 확인합니다.</li>
  <li>실무 최적화: 표지 기본값(4쪽) 선입력, 종이별 평량 매칭 기능으로 불필요한 단계를 줄였습니다.</li>
  <li>공유 기능: 계산된 사양을 텍스트로 복사하여 제작처나 디자이너에게 빠르게 전달할 수 있습니다. </li>
</ul>
</div>
<br>
<img src="/assets/tools_06.jpg" alt="6" />
<br>
<div class="text-narrow">
<h1>03. 책 크기 비교표</h1>
<ul>
  <li>인터랙티브 비교: 표 위에 마우스를 올리면 여러 판형의 크기를 실시간으로 비교해 볼 수 있습니다.</li>
  <li>데이터 큐레이션: 판형별 대표 서적의 정보(표지, 작가, 출판사, 규격)를 위계 있게 정리하여 기획 단계의 영감을 제공합니다.</li>
</ul>
</div>
<br>
<img src="/assets/tools_07.jpg" alt="7" />
<br>
<div class="text-narrow">
<h1>04. 글자 수 계산기</h1>
<ul>
  <li>다양한 기준: 공백 포함/제외 글자 수, 단어 수는 물론 출판 실무의 핵심인 '원고지 장수' 환산 기능을 제공합니다.</li>
  <li>업무 효율: 입력한 텍스트를 그대로 다시 복사할 수 있어 교정·교열 작업 중에도 끊김 없이 사용 가능합니다.</li>
</ul>
</div>

<div class="text-narrow">
<h1>05. QR코드 생성기</h1>
<ul>
  <li>안정성 최우선: 유효기간 제한이나 링크 변조 걱정 없는 안전한 QR코드를 생성합니다.</li>
  <li>전문가용 다운로드: 고해상도 인쇄를 위한 SVG(벡터) 파일과 웹용 PNG 파일을 모두 제공합니다.</li>
  <li>맞춤 가이드: 이미지 확장자가 낯선 편집자를 위해 상황별 추천 툴팁을 배치했습니다.</li>
</ul>
</div>

<div class="text-narrow">
<h1>06. 그 밖의 기능</h1>
<ul>
  <li>교정 부호 사용법: 헷갈리기 쉬운 교정 기호와 용례를 표로 정리해 실무 숙련도를 높여줍니다.</li>
  <li>근무/급여 관리: 근무 기간 계산기와 연도별 연봉 실수령액 확인 기능을 통해 경력 관리를 돕습니다.</li>
  <li>사이트 모음: 편집자가 업무 중 반드시 참고해야 할 유용한 사이트들을 한데 모았습니다.</li>
</ul>
</div>

<h1>디자인 및 UX 특징</h1>
<h3>01. 비주얼 중심의 직관적인 메뉴</h3>
<img src="/assets/tools_11.jpg" alt="11" />
<ul>
  <li>아이콘 메타포: 텍스트만으로 구성된 단조로운 메뉴 대신, 각 도구의 성격을 담은 아이콘 이미지를 활용하여 접근성과 심미성을 모두 높였습니다.</li>
  </ul>
<img src="/assets/tools_12.jpg" alt="12" />
<ul>
  <li>인터랙티브 피드백: 메뉴에 마우스를 올리면 아이콘이 부드럽게 확대되어, 현재 선택 중인 기능을 명확하게 인지할 수 있습니다.</li>
</ul>
<h3>02. 고민이 필요 없는 명확한 인터페이스</h3>
<img src="/assets/tools_13.jpg" alt="13" />
<ul>
  <li>상태 표시 최적화: 버튼 활성화 조건을 직관적으로 설계했습니다. 조건이 충족되지 않은 버튼은 'Disabled' 상태를 명확히 표시하여 불필요한 클릭을 방지합니다.</li>
</ul>
<img src="/assets/tools_14.jpg" alt="14" />
<ul>
  <li>아웃링크 안내: 외부 사이트로 연결되는 메뉴에는 별도의 아웃링크 아이콘을 부착하여 페이지 이탈을 미리 인지할 수 있게 배려했습니다.</li>
</ul>
<h3>03. 친절한 실시간 상태 안내</h3>
<img src="/assets/tools_15.jpg" alt="15" />
<ul>
  <li>토스트(Toast) 알림: 복사나 삭제 등 주요 기능을 실행할 때마다 하단에 토스트 메시지를 띄워, 작업이 성공적으로 완료되었음을 확실하게 안내합니다.</li>
</ul>
</div>

<div class="text-narrow">
<h1>브랜드 디자인</h1>
<h3>01. 디자인 메타포: 종이의 물성</h3>
<img src="/assets/tools_16.jpg" alt="16" />
<ul>
  <li>출판 편집자의 가장 본질적인 도구인 '종이'를 디자인 모티프로 삼았습니다. 평면적인 디지털 화면 속에서도 종이의 존재감을 느낄 수 있도록 요소를 2도 가량 회전시켜, 책상 위에 무심한 듯 놓인 원고의 느낌을 재현했습니다. 또한 끝이 접힌 듯한 디테일을 더해 종이의 입체감을 시각적으로 표현했습니다.</li>
  </ul>
<h3>02. 컬러 시스템: Classic Black & White</h3>
<img src="/assets/tools_17.jpg" alt="17" />
<ul>
  <li>에디터스 룸의 정체성에 맞추어 Black & White를 메인 컬러로 사용했습니다. 이는 화려한 장식보다 '텍스트'와 '콘텐츠' 그 자체에 집중하는 편집자의 전문성을 상징하며, 장시간 작업에도 눈이 피로하지 않은 실무 환경을 제공합니다.</li>
</ul>
<h3>03. 아이덴티티 시스템</h3>
<img src="/assets/tools_18.jpg" alt="18" />
<ul>
  <li>〈Tools〉의 로고와 에디터스 룸의 워드마크를 결합한 통합 아이덴티티 시스템을 구축했습니다. 이는 본 서비스가 에디터스 룸의 연장선에 있음을 보여주었습니다.</li>
</ul>
